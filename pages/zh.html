点匹配

2024

一个三消游戏，你可以通过对齐相同颜色的点来得分。使用 HTML Canvas 和 TypeScript 构建，使用简单的 NoSQL 数据库存储高分和功能请求。托管在 Google Cloud 上。

TypeScript Python Flask Cloud Run Firestore

湿球温度计算器

2023

一个单组件 Web 应用程序，它接受温度和湿度值，并返回“湿球温度”——身体通过蒸发冷却可以达到的温度。使用 Angular 构建，使用 Chroma.js 创建渐变。

Angular Chroma.js

日长可视化工具

2023

一个简单的 Web 应用程序，它显示一天中有多少是夜晚，多少是白天。允许用户更改时间和位置，以查看白天长度如何根据位置和时间变化。使用 Angular 构建，使用 Google Maps API 进行位置逻辑。

Angular Google Maps API

消息地图

2021 - 22

一个用于探索你的 Facebook Messenger 数据的工具。查看所有消息的总和，然后深入到特定消息。查看聊天、发送者等类别的摘要，并进行筛选。这是由两名法国学生开发的原始项目的一个分支。使用 Material UI 组件库重构为 Angular，使用 NgRx (Redux) 进行状态管理，使用 D3.js 进行数据可视化。

Angular Angular Material NgRx D3
