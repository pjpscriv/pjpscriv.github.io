*Les noms* sont des mots qui désignent des personnes, des lieux, des choses ou des idées. En français, chaque nom a un genre grammatical - masculin ou féminin - qui influe sur [les déterminants](../determinants), les adjectifs et les pronoms qui l'accompagnent.

## Noms communs et noms propres

*Les noms communs* désignent des choses ou des catégories générales : *un chat*, *une ville*.

*Les noms propres* désignent des personnes, des lieux ou des choses spécifiques et prennent une majuscule : *Montréal*, *le Québec*, *le Saint-Laurent*.

> **Note :** En français, les noms propres prennent moins souvent la majuscule qu'en anglais. Les jours de la semaine, les mois et les nationalités utilisées comme adjectifs ne prennent *pas* de majuscule - *lundi*, *janvier*, *un livre français*.

## Le Genre

Tous les noms français sont soit **masculins**, soit **féminins**. Pour les noms animés (personnes, animaux), le genre suit souvent le sexe naturel - *le père / la mère*, *le chat / la chatte* - mais pour les noms inanimés, il doit généralement être mémorisé. Apprendre l'article avec le nom (*le/la/un/une*) est l'approche la plus fiable.

Cela dit, la terminaison d'un nom est souvent un bon indice.

### Terminaisons masculines

- **-age** - le fromage, le garage, le voyage
  - <span style="opacity: 0.5">*exceptions : la page, la cage, l'image, la plage*</span>
- **-ment** - le gouvernement, le moment
  - <span style="opacity: 0.5">*exceptions : la jument*</span>
- **-eau** - le chapeau, le bateau, le cadeau
  - <span style="opacity: 0.5">*exceptions : l'eau, la peau*</span>
- **-eur** - le chanteur, le docteur
  - <span style="opacity: 0.5">*exceptions : la fleur, la chaleur, la peur, la couleur, la douleur, la hauteur*</span>
- **-in** - le vin, le jardin, le matin
  - <span style="opacity: 0.5">*exceptions : la fin, la main, la faim*</span>
- **-on** - le balcon, le crayon
  - <span style="opacity: 0.5">*exceptions : la maison, la saison, la raison, la chanson, la leçon, la prison*</span>
- **-et** - le jouet, le billet
- **-ier** - le papier, le fermier
- **-isme** - le tourisme, le capitalisme

### Terminaisons féminines

- **-tion / -sion** - la nation, la révolution, la passion
- **-ité** - la liberté, la qualité, la curiosité
- **-ure** - la voiture, la nature, la culture
  - <span style="opacity: 0.5">*exceptions : le murmure, le mercure*</span>
- **-ance / -ence** - la chance, la danse, la violence, la patience
  - <span style="opacity: 0.5">*exceptions : le silence*</span>
- **-ie** - la boulangerie, la géographie
  - <span style="opacity: 0.5">*exceptions : le génie, le parapluie*</span>
- **-ette** - la baguette, la serviette
  - <span style="opacity: 0.5">*exceptions : le squelette*</span>
- **-esse** - la princesse, la richesse
- **-euse** - la chanteuse, la danseuse
- **-ière** - la frontière, la cuisinière
  - <span style="opacity: 0.5">*exceptions : le derrière, le cimetière*</span>
- **-ée** - la journée, la pensée
  - <span style="opacity: 0.5">*exceptions : le lycée, le musée, le trophée*</span>
