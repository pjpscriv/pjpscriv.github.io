*Les adjectifs* sont des mots qui décrivent ou modifient [les noms](../nouns). En français, les adjectifs doivent **s'accorder** en genre et en nombre avec le nom qu'ils qualifient, et leur position par rapport au nom suit des règles précises.

## L'Accord

### Le Genre

La forme de base d'un adjectif français est le masculin singulier. Le féminin se forme généralement en ajoutant **-e** :

- **petit → petite**
- **grand → grande**
- **français → française**

Quand le masculin se termine déjà par **-e**, la forme est identique pour les deux genres :

- **rouge** *(un ballon rouge / une voiture rouge)*
- **jeune**

De nombreux adjectifs suivent des modèles selon leur terminaison :

- **-eux → -euse** - heureux → heureuse, sérieux → sérieuse
- **-er → -ère** - premier → première, étranger → étrangère
- **-f → -ve** - actif → active, neuf → neuve
- **-on → -onne** - bon → bonne, breton → bretonne
- **-en → -enne** - ancien → ancienne, européen → européenne
- **-el → -elle** - naturel → naturelle, officiel → officielle
- **-et → -ète** - discret → discrète, concret → concrète
  - <span style="opacity: 0.5">*exceptions : muet → muette, net → nette*</span>

Certains adjectifs courants sont entièrement irréguliers :

- **beau → belle**
- **nouveau → nouvelle**
- **vieux → vieille**
- **blanc → blanche**
- **doux → douce**
- **faux → fausse**

### Le Nombre

Le pluriel se forme généralement en ajoutant **-s** au singulier. Les deux genres suivent cette règle :

- **petit → petits**, **petite → petites**

Quand le singulier se termine déjà par **-s** ou **-x**, le pluriel est identique :

- **gris → gris**
- **heureux → heureux**

Certains adjectifs suivent d'autres modèles :

- **-eau → -eaux** - beau → beaux, nouveau → nouveaux
- **-al → -aux** - national → nationaux, royal → royaux
  - <span style="opacity: 0.5">*exceptions : fatal → fatals, banal → banals, naval → navals*</span>

## La Position

La plupart des adjectifs français se placent **après** le nom qu'ils qualifient :

- *une voiture **rouge***
- *un homme **intelligent***

Cependant, un groupe d'adjectifs courants se place **avant** le nom. On peut les retenir avec le sigle **BÂNGS** :

- **B**eauté : *beau, joli*
- **Â**ge : *jeune, vieux, ancien*
- **N**ombre/ordre : *premier, dernier, prochain, deuxième*
- **G**randeur : *grand, petit, gros, long, haut*
- **S**upériorité : *bon, mauvais*

> **Note :** certains adjectifs changent de sens selon leur position. Avant le nom, *ancien* signifie « d'autrefois » (*mon ancien professeur*) ; après, il signifie « vieux » (*un bâtiment ancien*). De même, *grand* avant le nom signifie « remarquable » (*un grand homme*) ; après, il signifie « de grande taille » (*un homme grand*).

## Les Formes de Liaison

<u>Trois adjectifs</u> ont une forme masculine singulière supplémentaire utilisée devant une voyelle ou un *h* muet :

- **beau → bel** - *un bel homme*
- **nouveau → nouvel** - *un nouvel appartement*
- **vieux → vieil** - *un vieil arbre*

Ces formes n'affectent que le masculin singulier. Les formes féminines (*belle*, *nouvelle*, *vieille*) restent inchangées.

## Les Invariables

Certains adjectifs ne s'accordent pas avec le nom :

- Les adjectifs de couleur dérivés de noms : **marron**, **orange**, **crème**, **saumon**
  - <span style="opacity: 0.5">*contraste : rouge, vert, bleu, jaune etc. s'accordent normalement*</span>
- Les couleurs composées (deux mots décrivant une même nuance) : *bleu marine*, *vert foncé*, *rouge vif*

## La Comparaison

### Le Comparatif

Pour comparer des adjectifs, on place **plus**, **moins** ou **aussi** avant l'adjectif et **que** après :

- **plus ... que** → +
- **moins ... que** → -
- **aussi ... que** → =

### Le Superlatif

Le superlatif utilise un article défini + **plus** ou **moins**. L'article s'accorde avec le nom :

- **le/la/les plus ...**
- **le/la/les moins ...**

*C'est la ville **la plus belle** du pays.*

Quand l'adjectif suit le nom, l'article défini est répété : *le film **le plus** intéressant*.

### Les Formes Irrégulières

Quelques adjectifs ont des formes comparatives et superlatives irrégulières plutôt que d'utiliser *plus/moins* :

- **bon → meilleur(e)**
- **mauvais → pire**
  - <span style="opacity: 0.5">*plus mauvais existe aussi mais est moins courant*</span>
